# 需求梳理网站 · 项目说明

**日期**：2025-02-27  

**位置**：本仓库下的 `requirement-website/` 文件夹，与 `web-demo`、小程序等其它项目分开存放，避免混淆。

## 是什么

- 一句话需求 → 澄清问题（可跳过）→ 生成创业级 PRD → 支持标注修订与导出。
- 前端：`requirement-website/frontend`（React + Vite + TypeScript）
- 后端：`requirement-website/backend`（Node + Express，调大模型 API）

## 今天做到哪了

- 后端已补全（server、llm、prompts、.env 配置）
- 澄清 5 问展示与生成流程已跑通
- 生成文档、错误提示、加载动画、快速生成（FAST_GENERATE）已就绪
- API Key 在 `backend/.env`，启动顺序：先起后端再起前端

## 启动方式

```bash
# 终端1：后端
cd requirement-website/backend && npm run start

# 终端2：前端
cd requirement-website/frontend && npm run dev
```

浏览器打开前端显示的地址（如 http://localhost:5173）。
